# Retail Sales Analysis and Sales Prediction

## Overview

This project performs exploratory data analysis and sales prediction using a retail sales dataset.

## Objectives

- Understand customer purchasing behavior.
- Analyze sales and profit trends.
- Visualize important business insights.
- Build machine learning models for sales prediction.

## Dataset

Sample Superstore Dataset

Features include:

- Sales
- Profit
- Quantity
- Discount
- Region
- Category
- State
- Customer Segment

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Models

- Linear Regression
- Random Forest Regressor

## Evaluation Metrics

- MAE
- MSE
- RMSE
- R² Score

## Visualizations

- Sales by Category
- Sales Trend
- Heatmap
- Profit Analysis
- Pairplot
- Histogram
- Boxplot

## Business Insights

- Technology category had the highest sales.
- Discounts reduced profitability.
- West region generated the highest profit.
- Random Forest outperformed Linear Regression.
